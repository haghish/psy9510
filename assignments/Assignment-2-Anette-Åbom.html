Character strings

An object can be displayed by using the print function print().

print("Day 2 of the Introduction to Statistics with R")
## [1] "Day 2 of the Introduction to Statistics with R"

A string is a collection of character variables. Letâ€™s say I have a string called mystring which consists of the 3 parts:

mystring = 'guitar drums piano'

I can break this string by defining the separators.

strsplit(mystring, "guitar drums piano", split = " ")
## [[1]]
## [1] "guitar" "drums"  "piano"

Multiple strings can be stored and combined in a character vector, which is created with the c() function.

c("guitar", "drums", "piano")
## [1] "guitar" "drums"  "piano"

If I want to manipulate a whole string within the character vector, I can locate it, and then change it.

a = c("guitar", "drums", "piano")
a[1] = "bass"
a
## [1] "bass"  "drums" "piano"

I now want to have a look at my vector a, and figure out the length of the strings.

str_length(a)
## [1] 4 5 5

I also want to only see the first three characters within the strings.

substr(a, 1, 3)
## [1] "bas" "dru" "pia"

Finally, I want to know which strings contain a particular character.

str_subset(a, "[s]")
## [1] "bass"  "drums"

We were recommended to start learning and use RegExplain since this is considered being a good tool to use for complex search for a string.

Loading and saving data

RDS is used for saving and loading only a single object. This can be done by writing the code saveRDS() or readRDS(). When saving the object, the object comes as the first argument and the file as the second.

Data is quite often stored as CSV files (Comma Separated Values). We can use the read.csv() function to read those as a data frame. Likewise, we can read an excel file by using the function read_excel(). It is possible to specify a worksheet by name and number. For example: read_excel(xlsx_example, sheet = "chickwts") or read_excel(xlsx_example, sheet = 4).

To load a dataset we use the data( ) function. Some data sets are available in R for everyone, like for example the iris data set.

I will use the package â€œpanderâ€ when analyzing the data set.
